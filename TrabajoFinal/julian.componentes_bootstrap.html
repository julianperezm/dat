Introducción

Para comenzar vamos a hacer un pequeño recordatorio de qué es bootstrap y de los conceptos que hemos dado en la asignatura sobre este.

¿Qué es Bootstrap?

Bootstrap es un framework de código libre que ayuda a desarrollar aplicaciones web con una interfaz limpia y sencilla, sus ventajas son varias, desde la compatibilidad con distintas plataformas hasta un comportamiento responsivo de los navegadores con la ayuda de su sistema GRID.

Conceptos dados en la asignatura

En la asignatura comenzamos con las características más importantes de bootstrap, como son el uso del viewport virtual o el comportamiento resposive con el uso de Grid. También vimos la herramienta que se utiliza para la detección de algunos errores en el html de diseños de bootstrap, llamada bootlint. Tras esto comenzamos con el estudio tanto teórico como práctico de algunos de los componentes más importantes de bootstrap, como pueden ser los paneles, botones, nav o dropdowns.

El objetivo de este guion es aumentar los conocimientos sobre los componentes de bootstrap y seguir trabjando con estos. Nos centraremos en la versión más reciente de bootstrap, la versión 4.5

Badge


Gracias a diferentes componentes bootstrap podemos resaltar de diferente forma los elementos de nuestro html, uno de estos componentes es 'badge', este consiste en un pequeño recuadro cerca del correspondiente elemento con el texto que tú le atribuyas o con el texto que quieras mostrar al lector de manera llamativa. A continuación podrás ver los usos más destacados de este componente.

Más información >


Básico

Si queremos añadir información adicional a cualquier contenido de manera básica utilizando este componente nos valdría con añadir la siguiente clase:

class = "badge badge-secondary"

Esta clase la podemos insertar a diferentes elementos del html como h1, h2, etc. Del mismo modo que aumenta el tamaño del texto aumenta el badge.

Collapse


El componente collapse es una opción muy interesante si lo que buscas es mostrar/ocultar contenido en tu página web, este componente es bastante útil si no quieres sobrecargar tu página con tanta información que sature al lector, también se puede llevar a otros ámbitos como el ocultar botones, enlaces, etc.

Más información >

Básico

Si creamos diferentes elementos podemos crear un collapse de una forma simple tan solo añadiendole la clase:

data-toggle="collapse"

Debes crear el mismo id para el elemento y el contenido que quieres mostrar/ocultar.

Pagination


En el caso de que nuestra página sea tan extensa que queramos utilizar distintas páginas o tengamos una sección que sea tan pesada visualmente que queramos ordenarlas en diferentes páginas y necesitemos navegar facilmente por estas, Bootstrap tiene un componente que nos permite realizar esta acción de manera sencilla llamada pagination.

Básico

Popovers


En Bootstrap tenemos un componente que nos permite añadir información adicional a nuestros elementos creados. Esto resulta útil cuando quieres dar una pequeña información sobre un elemento y no ocupar espacio en la web. Esta información puede ser algún tip o alguna instrucción oculta relacionada con este, ya que los popovers se activan al pulsar en ellos.

Más información >

Básico

Si queremos usar de manera básica la clase popover para añadir texto sobre algún elemento de nuestro html se utilizaría con el atributo:

data-toggle="popover"

Si queremos utilizar el componente de este modo debemos añadir a nuestro html un script con el siguiente código:

$(function () {
   $('[data-toggle="popover"]').popover()
})

Una de las ventajas que tiene este componente es que lo podemos añadir a diferentes elementos como pueden ser botones, enlaces, etc.


Título, Deshabilitado y Más





Cuidado!

Podemos añadir títulos a nuestro popover mediante el atributo:

title="titulo de prueba"

Otra opción bastante interesante es deshabilitarlo, con el uso del atributo:

data-content="Disabled popover"

En el caso de que nuestro popover sea un botón podemos cambiar el contexto como en muchos otros componentes con la clase:

class="btn btn-warning"

Para este caso tendremos que utilizar el elemento 'a' y no el 'button' para un mejor comportamiento y debemos añadir el atributo:

tabindex="0"

Progress


Una barra de progreso en tu página web puede ser interesante en distintos aspectos, como comprobar la seguridad de unas contraseñas, o contabilizar el porcentaje de progreso de cualquier acción de tu página. Todo esto y mucho más lo podemos conseguir gracias a este componente.

Más información >

Básico

Las barras de progreso se pueden utilizar de una manera básica con tan solo añadir la clase:

class="progress"

Si queremos llenarla al 50%, por ejemplo, lo podemos hacer utilizando:

class="progress-bar" role="progressbar" style="width: 50%" aria-valuenow="50" aria-valuemin="0" aria-valuemax="100"

Otra opción bastante interesante es añadir una etiqueta de progreso en la misma barra. Se haría añadiendo texto en el div en el que has indicado que vas a crear la barra de progreso.


Diferentes Aspectos

Podemos dar diferentes aspectos a la barra de progreso, según lo que queramos expresar, utilizando las variaciones contextuales como en los componentes anteriores.

class="progress-bar bg-info/danger/warning"

Otra opción es utilizar estos estilos en una misma barra. En este caso iremos cambiando el atributo style según la cantidad que queramos revelar.

style="width: 30%"

Barras Rayadas





Una forma de cambiar el aspecto, aparte de los colores, puede consistir en añadir rayas a la barra y esto se puede realizar añadiendo la clase:

class="progress-bar progress-bar-striped"

Podemos hacer que estas rayas se muevan y hacer un diseño más animado añadiendo la clase:

class="progress-bar progress-bar-striped progress-bar-animated"

Spinner


El componente spinner en bootstrap permite informar al usuario de manera gráfica sobre un proceso que se está realizando en el momento, como la petición de un recurso o el cambio a otra web.

Más información >

Básico

El uso más básico del componente spinner de bootstrap se realiza añadiendo a nuestro elemento la clase:

class="spinner-border"

Para eliminar el texto de nuestro elemento debemos añadir a este la clase:

class="sr-only"

Utilizando diferentes variaciones contextuales podemos cambiar el color de nuestro spinner, utilizando las clases:

class="spinner-border text-primary/info/warning/danger"

Cambio de spinner

Podemos cambiar el diseño del spinner utilizando una clase diferente a la anterior pero del mismo modo, la clase es:

class="spinner-grow"

En este caso al igual que el anterior podemos cambiar de color del mismo modo, con las clases:

class="spinner-grow text-primary/info/warning/danger"

Dentro de elementos



Podemos meter los spinners en diferentes elementos como pueden ser botones, dentro del elemento "button" incluiremos un span con los mismos datos que antes:

class="spinner-border"

En este caso podemos ocultar el texto o no con el atributo que hemos visto antes.

Y al igual si queremos cambiar el icono del spinner:

class="spinner-grow"
